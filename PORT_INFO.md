# 포트 설정 정보

## 현재 포트 설정

이 프로젝트는 **포트 3002**에서 실행됩니다.

- **정기구독 해지 방어기**: `http://localhost:3002`
- **다른 프로젝트**: `http://localhost:3000` (기본값)

## 접속 방법

브라우저에서 다음 URL로 접속하세요:
```
http://localhost:3002
```

## 포트 변경 방법

다른 포트를 사용하고 싶다면 `package.json` 파일을 수정하세요:

```json
"scripts": {
  "dev": "next dev -p 3002",  // 원하는 포트 번호로 변경
  "start": "next start -p 3002"
}
```

그리고 서버를 재시작하세요.

## 두 프로젝트 동시 실행

두 프로젝트를 동시에 실행하려면:
- 프로젝트 1: `http://localhost:3000`
- 프로젝트 2: `http://localhost:3002` (이 프로젝트)

각각 다른 포트를 사용하므로 충돌 없이 동시에 실행할 수 있습니다.
